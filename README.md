# Online Examination System

## Project Overview
A console-based Online Examination System developed using Core Java, JDBC, and MySQL. The system allows administrators to create exams and manage questions, while students can attempt exams and receive instant score evaluation.

## Technologies Used
- Core Java
- JDBC
- MySQL

## Features
- Admin Login
- Add Exam
- Add Questions
- Student Exam Module
- Score Calculation

## Architecture
MainApp → Service → DAO → MySQL Database
